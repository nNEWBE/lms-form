# Metro LMS Registration Form

A state-of-the-art, high-fidelity library member registration form featuring a modern design, dynamic interactive elements, and custom picker widgets.

## Features
- **Premium Compact UI**: Custom dark solid background layout with thin borders and zero shadow glow.
- **Dynamic 3D Member Card**: Real-time syncing from input fields to a simulated digital ID card that tilts in 3D on hover.
- **Custom Picker Suite**: Handcrafted custom date pickers (Date of Birth, Start Date, Expiry Date) with smooth scale entry, custom navigation, and outlined actions, alongside a full-width select dropdown.
- **tactile Micro-interactions**: Fluid active track range slider fills, smooth input focus, loading spinners, and validation wobble feedback.
